# Quick Setup

## Requirements

- Node.js v16+
- npm
- Expo CLI

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Then choose:
- `a` for Android
- `i` for iOS
- `w` for Web

## Issues?

```bash
# Clear cache
npm start -c

# Reinstall
rm -rf node_modules package-lock.json
npm install
```
